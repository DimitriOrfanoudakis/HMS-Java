# Hotel Management System

# 🇩🇪🇦🇹🇨🇭 DE


## Beschreibung
Hier entsteht ein einfaches Hotelverwaltungsprogramm. Zuerst als kleines Übungsprojekt in Python mit Dictionaries und .json Dateien zum speichern umgesetzt.

Nun wird versucht, es nochmal stukturierter und mit Objektorientierter Programmierung zu implementieren. 
Durch Modularisierung soll diesmal eine leichtere Wartbarkeit des Programms erzielt werden und 
eine zukünftige Implementierung zusätzlicher Funktionen erleichtert werden.

Zur Speicherung wird eine PostgreSQL Datenbank verwendet. SQL Statements und Queries sollen in den DAO Klassen gekapselt werden,
damit die Datenbank theoretisch ausgetauscht werden kann.
